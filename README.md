# Week 1: Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Online Retail II dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Dataset Overview
- Missing Values Analysis
- Duplicate Rows Analysis
- Top 10 Best-Selling Products
- Top 10 Revenue-Generating Products
- Country-wise Sales Analysis
- Monthly Sales Trend
- Correlation Heatmap
- Outlier Detection using Box Plots

## Key Insights
- The United Kingdom generated the highest revenue.
- November recorded the highest monthly sales.
- Quantity has a stronger relationship with revenue than price.
- The dataset contains missing values, duplicate records, and outliers.
